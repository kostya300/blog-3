# Блог на Django(В стадии разработки)

Простой и функциональный блог с публикациями, комментариями, тегами и картой сайта. Проект создан на Django и демонстрирует базовые возможности веб‑приложения для ведения онлайн‑дневника или корпоративного блога.

## Функциональность

Проект включает следующие возможности:

* **Публикация постов** с заголовком, текстом, автором, датой публикации и статусом (черновик/опубликовано).
* **Тегирование** постов через `django-taggit` — удобная категоризация контента.
* **Комментарии** к постам с модерацией (активные/неактивные).
* **Пагинация** списка постов (3 поста на странице).
* **Популярные статьи** в сайдбаре — топ‑5 постов по количеству комментариев.
* **Счётчик просмотров** — показывает, сколько раз открыли пост.
* **Карта сайта (sitemap.xml)** — для индексации поисковиками.
* **Загрузка изображений** к постам с валидацией формата и размера (до 5 МБ, JPG/PNG/WebP).
* 
* **Кол-просмотров**
![Скриншот интерфейса блога](https://github.com/kostya300/blog-3/blob/main/Снимок%20экрана%202026-02-27%20191458.png)
* **контекстные процессоры для sidebar.html во избежание дублирования**

## Технологии и зависимости

* Python 3.13+
* Django 6.0.2
* Pillow (для работы с изображениями)
* django-taggit (управление тегами)
* Django Sites Framework (для карты сайта)
